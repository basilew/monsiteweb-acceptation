Bridge bidding guide
Inspired by the American Standard

These basic charts are helpful to refresh memory. One must keep in mind that context judgment overrides any bid, because any grid will always be incomplete - like these which don't take opponent's bid into account.

In each table, bids are ordered from the most to the least preferred, if hesitating.

This system uses High Card Points (HCP) (aces=4, kings=3, queens=2, jacks=1). Use your judgment to tweak.
